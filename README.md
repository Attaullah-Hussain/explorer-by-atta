# ✨ Explorer by Atta
A full-stack, enterprise-grade Data Analytics & Machine Learning SaaS platform.

### 🚀 Try it live here: [Click here to launch the App](https://explorer-by-atta.streamlit.app)

### 🛠️ Key Features:
* **Live API Connectors:** Pulls real-time financial market data via Yahoo Finance.
* **Machine Learning Engine:** Uses Scikit-Learn (Linear Regression) to forecast future data trends.
* **AI Chat Assistant:** Integrated with Groq Llama 3.3 for conversational data analysis.
* **Automated Data Cleaning:** One-click duplicate removal, missing value handling, and column renaming.
* **Admin God Mode:** Secure SQLite backend with admin dashboard and server storage controls.
* **Enterprise Architecture:** Modular, multi-file codebase separating UI, Authentication, and Backend Logic.
