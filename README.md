## 💎 Platform Core Features

| **🧠 AI Analyst** | Powered by **Groq Llama 3.3 API** | Conversational data insights & natural language queries. |

| **🔐 Secure Auth** | **SQLite** + **Bcrypt** Hashing | Enterprise-grade user management and data isolation. |

---

## 📂 Engineering Architecture
Designed with a **Modular Design Pattern** for high performance and maintainability:

* 🛠️ **`app.py`** : The UI Orchestrator & Main Application Logic.
* 🛡️ **`auth.py`** : Security Layer (SQL DB Management & Encryption).
* 📁 **`backend.py`** : Core Processing (Workspaces, OS handling & PDF Exports).
* 🎨 **`styles.py`** : Visual Framework (Neon-Vector CSS & Dark/Light Themes).
* 📋 **`requirements.txt`** : Cloud environment dependency mapping.
